Thymeleaf é um moderno mecanismo de modelo Java do lado do servidor para web e independente ambientes. Com módulos para o Spring Framework, uma série de integrações com suas ferramentas favoritas e a capacidade de conectar em sua própria funcionalidade, Thymeleaf é ideal para os dias modernos Desenvolvimento web HTML5 JVM - embora haja muito mais que pode Faz.
WebJars estão do lado do cliente bibliotecas da Web (por exemplo, jQuery; Bootstrap) empacotadas em JAR (Java Arquivar) arquivos.
Hibernate ORM habilita desenvolvedores para escrever mais facilmente aplicativos cujos dados sobrevivem o processo de candidatura. Como um mapeamento objeto / relacional (ORM) framework, o Hibernate está preocupado com a persistência de dados aplica-se a bancos de dados relacionais (via JDBC).
Além de sua própria API "nativa", o Hibernate também é uma implementação do Especificação da Java Persistence API (JPA). Como tal, pode ser facilmente usado em qualquer ambiente que suporte JPA, incluindo Java SE aplicativos, servidores de aplicativos Java EE, Enterprise OSGi recipientes, etc.
Bootstrap é um aberto kit de ferramentas de origem para desenvolvimento com HTML, CSS e JS. Rapidamente prototipar suas idéias ou construir seu aplicativo inteiro com nosso Sass variáveis ​​e mixins, sistema de grade responsivo, extenso pré-construído componentes e poderosos plugins construídos em jQuery.
