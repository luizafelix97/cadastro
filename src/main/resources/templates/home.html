Hibernate ORM habilita desenvolvedores para escrever mais facilmente aplicativos cujos dados sobrevivem o processo de candidatura. Como um mapeamento objeto / relacional (ORM) framework, o Hibernate está preocupado com a persistência de dados aplica-se a bancos de dados relacionais (via JDBC).
Além de sua própria API "nativa", o Hibernate também é uma implementação do Especificação da Java Persistence API (JPA). Como tal, pode ser facilmente usado em qualquer ambiente que suporte JPA, incluindo Java SE aplicativos, servidores de aplicativos Java EE, Enterprise OSGi recipientes, etc.
